# create-su-app 

# Usage

```zsh
npx create-su-app <dir> --template <template-name>
```

Example

```zsh
npx create-su-app new-project --template primsa-graphql 
```

# Available Templates

```bash
├── express-typescript
├── prisma
├── prisma-graphql
├── react_fe
├── server-js
├── snowpack-express
└── snowpack-typescript
```
